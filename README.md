- **GameManager**: 全局状态管理
- **AudioManager**: 音频系统

### 难度曲线
```gdscript
敌人血量 = 基础血量 * (1 + 波次 * 0.5)
敌人速度 = 基础速度 + 波次 * 10
Boss血量 = 基础血量 * (1 + 波次 * 0.3)
敌人数量 = 20 + (波次 * 5)
```
